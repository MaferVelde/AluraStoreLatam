# Alura Store Latam

Este proyecto ha sido creado para analisar la información que tenemos de 4 tiendas propiedad del Sr. Juan, con el propósito de conocer el desempeño de cada tienda y así poder elegir la menos redituable para su venta.

## Tecnologías utilizadas:

* Python a travez de Google colab

## Instalación y uso

No es requerida instalación, para ejecutar el programa es necesario crear una cuenta de Google , entrar a colab y abrir el archivo.
Al entrar encontramos la información dividida en varios puntos, de esa manera es mas sencillo observar y coprender paso a paso nuestra elección final.
